
## Custom knowledge ChatBot

### requirements

pip >= 22.0.2 & python >= 3.10.12

create and .env with apenAI api key, 

```bash
  python3 -m venv .venv
  source .venv/bin/activate
  pip3 install -r requirements.txt
```

```bash
    source .venv/bin/activate
    python3 main.py
```
